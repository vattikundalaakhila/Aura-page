# 💍 Aura – Smart Ring Landing Page

Aura is a modern and responsive Single Page Application (SPA) built using React and Tailwind CSS.  
It represents a smart wearable ring designed to track health metrics such as heart rate, sleep patterns, activity levels, and more.

This project was created for frontend practice and UI development improvement.

## 📦 Installation

1. Clone the repository:

git clone https://github.com/vattikundalaakhila/Aura-page.git

2. Navigate into the project folder:

cd react-page

3. Install dependencies:

npm install

4. Start development server:

npm run dev
